## ⚠️ Challenges Faced

During development, several challenges were encountered:

- Configuring secure communication between private EC2 instances.
- Debugging Security Group and Route Table issues.
- Configuring NFS shared storage.
- Implementing LUKS encryption.
- Configuring IAM Roles for secure AWS access.
- Debugging FastAPI service deployment using systemd.
- Integrating API Gateway with private resources using VPC Link.
- Configuring Auto Scaling and health checks.
- Monitoring infrastructure using CloudWatch.
