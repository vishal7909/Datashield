# Service Layer

## Overview

The Service Layer manages metadata generated by the Analyzer Service. It stores metadata in Amazon RDS and provides APIs for retrieving data through the dashboard.

## Responsibilities

- Receive metadata from Analyzer
- Store metadata in Amazon RDS
- Retrieve metadata
- Provide dashboard APIs
- Display processed file information

## Technology Stack

- Python
- FastAPI
- Uvicorn
- SQLAlchemy
- PyMySQL
- Jinja2

## API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | / | Dashboard |
| GET | /health | Health check |
| POST | /metadata | Store metadata |
| GET | /files | Retrieve metadata |

## Deployment

```bash
pip install -r requirements.txt
uvicorn app:app --host 0.0.0.0 --port 8000
```

## Systemd Service

```bash
sudo systemctl start datashield
sudo systemctl status datashield
```
